## 1. 🍅 뽀모도로 RPG 앱 (1순위)

### 상태
- **현재 개발 중** (2025.10~12 목표)
- Flutter + Hive (로컬 저장)

### 핵심 기능
- 뽀모도로 타이머 (25분 집중 / 5분 휴식)
- RPG 캐릭터 성장 시스템
- 집중 완료 → 캐릭터 스탯 증가 (지능/건강/미모/사회성)
- 업적 시스템, 통계 대시보드

### 차별화 포인트
- ADHD 친화적 (게임화로 동기부여)
- 시간 기반이 아닌 **완료 기반** 목표
- "3시간 공부" ❌ → "타이머 6회 완료" ✅

### 기술 스택
```
Frontend: Flutter
Storage: Hive (로컬 DB)
알림: flutter_local_notifications
```

### 수익 모델
- Freemium: 무료 + 프리미엄 $2.99/월
- 프리미엄: 캐릭터 커스텀, 상세 통계, 위젯
- 예상 수익: 월 20~60만원 (출시 후 6개월)

### 개발 일정
```
Week 1-4: 타이머 기능 + 기본 UI
Week 5-6: 캐릭터 시스템
Week 7-8: 통계 + 알림
Week 9-12: UI 완성 + 테스트 + 출시
```

### 왜 1순위?
- ✅ 가장 단순 (Flutter 입문용)
- ✅ 서버 비용 0원
- ✅ 혼자 100% 가능
- ✅ 완성 가능성 90%
- ✅ 직접 매일 쓸 앱

## 2. 📍 지오펜싱 아바타 앱 (3순위)

### 상태
- 아이디어 단계
- 뽀모도로 완성 후 개발 예정

### 핵심 컨셉
**"현실 생활 = 게임 성장"**
- 특정 장소 방문 → 아바타 자동 성장
- 카페 도착 → "집중력 +5"
- 헬스장 도착 → "체력 +10"
- 마트 방문 → "생활력 +3"

### 차별화 포인트
- GPS 실시간 추적 ❌ (배터리 소모)
- **지오펜싱** ✅ (장소 도착 시에만 트리거)
- 갤럭시 루틴처럼 백그라운드 동작
- 사용자가 아무것도 안 해도 성장

### 기술 스택
```
Frontend: Flutter
Backend: Firebase
위치: Geofencing API
지도: Google Maps
푸시: Firebase Cloud Messaging
```

### 수익 모델
- Freemium: $4.99/월
- 광고 제거: $9.99 일회성
- 예상 수익: 월 50~150만원 (1년 후)

### 개발 일정
```
Week 1-4: 지오펜싱 + 푸시 알림
Week 5-8: 아바타 시스템 + 스탯
Week 9-12: 이벤트/업적 시스템
Week 13-16: 테스트 + 출시
```

### 리스크
- ⚠️ 배터리 최적화 어려움
- ⚠️ iOS/Android 동작 차이
- ⚠️ 위치 권한 거부 시 앱 사용 불가

### 왜 2순위?
- ✅ 차별화 최고 (경쟁 앱 없음)
- ✅ 재미 요소 최고
- ✅ 수익성 높음
- ⚠️ 뽀모도로로 Flutter 익힌 후 도전

## 3. ⏱️ 타임블록 + 트리거 앱 (2순위)

### 상태
- 아이디어 단계
- ADHD 특화 시간관리 앱

### 핵심 기능

**메인: 타임블록**
- 하루 일정을 블록 단위로 시각화
- "09:00-11:00 앱 개발" 같은 큰 작업

**서브: 트리거 시스템**
- 빈 시간에 5분짜리 작은 활동 추천
- 29개 트리거 라이브러리:
  - 자기계발: 영어, 글쓰기, 그림
  - 건강: 스트레칭, 워킹패드
  - 생활: 청소, 요리 준비
  - 경제: 투자 체크, 리뷰 작성

### 차별화 포인트
```
기존 앱들:
- Todoist: 할 일 관리만
- Google Calendar: 일정만
- Sunsama: $20/월 (비쌈)

이 앱:
타임블록 + 마이크로 습관 = 블루오션
```

### ADHD 친화 설계
- 빈 시간 빨간색 강조 (유튜브 방지)
- "지금 뭐 하지?" 버튼 (랜덤 추천)
- 5분 타이머
- 완료 시 애니메이션 + 사운드

### 기술 스택
```
Frontend: Flutter
Storage: Hive
UI: 커스텀 캘린더 위젯
```

### 수익 모델
- 무료: 트리거 10개, 분야 3개
- 프리미엄 $4.99/월: 무제한 + 통계
- 예상 수익: 월 30~80만원

### 왜 3순위?
- ✅ 직접 매일 쓸 앱 (ADHD 관리)
- ✅ 포트폴리오 가치 높음
- ✅ 서버 필요 없음
- ⚠️ UI/UX 복잡도 높음

## 4. 👨‍👩‍👧‍👦 가족 다이어리 앱 (4순위) ⭐NEW

### 상태
- 아이디어 단계 (2025.12 추가)
- Firebase 학습 후 개발 가능

### 핵심 컨셉
**"같이 안 사는 가족을 위한 프라이빗 SNS"**
- 카톡 가족방의 상위호환
- 글/사진/영상 타임라인
- 가족만 보는 비공개 공간

### 타겟
- 부모님 시골 계시는데 손주 사진 보여드리고 싶음
- 형제자매 각자 살면서 근황 공유
- 명절 아니면 연락 안 하는 가족들

### 기존 앱과 차별점

| 카톡 가족방 | 가족 다이어리 |
|------------|--------------|
| 사진 묻힘 | 타임라인으로 정리 |
| 알림 폭탄 | 하루 1번 요약 |
| 대화 위주 | 기록 위주 |
| 정리 안 됨 | 앨범 자동 정리 |

### 추가 기능 아이디어
- "오늘 뭐했어?" 하루 한 번 알림 (부담 없이 한 줄)
- 가족 캘린더 (생일, 기념일 자동 알림)
- 추억 회상 ("3년 전 오늘 엄마가 올린 글")
- 음성 메시지 (타이핑 어려운 어르신용)

### 기술 스택
```
Frontend: Flutter
Backend: Firebase
- Auth: 로그인/회원가입
- Firestore: 글/댓글 저장
- Storage: 사진/영상
- Cloud Messaging: 푸시 알림
```

### 수익 모델
- 무료: 기본 기능
- 프리미엄 $3.99/월: 무제한 저장, 포토북 출력
- 예상 수익: 월 20~50만원

### 개발 일정
```
Week 1-2: Firebase 학습
Week 3-6: 인증 + 그룹 시스템
Week 7-12: 포스팅 + 댓글
Week 13-16: 알림 + 앨범
Week 17-20: 테스트 + 출시
```

### 리스크
- ⚠️ 가족 전원이 깔아야 의미 있음
- ⚠️ 부모님 세대 앱 설치 유도 어려움
- ⚠️ 경쟁 앱: 패밀리앨범, 베이비타임, 밴드

### 왜 4순위?
- ✅ 경쟁 적음 (육아가 아닌 "가족" 타겟)
- ✅ 구독 모델 자연스러움
- ✅ 글로벌 확장 가능
- ⚠️ 백엔드 필요 (Firebase)
- ⚠️ 온보딩 허들 높음

## 5. 🛒 쇼핑 스크랩북 앱 (5순위)

### 상태
- 아이디어 단계
- 기술 난이도 높아 후순위

### 핵심 기능
- 여러 쇼핑몰 링크 한 곳에 저장
- AI 자동 카테고리 분류
- 가격 변동 추적 + 최저가 알림
- 친구/가족과 위시리스트 공유

### 차별화 포인트
- 네이버/쿠팡/알리 등 통합 관리
- 가격 히스토리 그래프
- 쇼핑 예산 관리 기능

### 기술 스택
```
Frontend: Flutter
Backend: Firebase Functions + Python
크롤링: 쇼핑몰별 파서
AI: OpenAI API (상품명 분류)
제휴: 쿠팡파트너스 API
```

### 수익 모델
- 1차: 제휴 수익 (링크 자동 전환)
- 2차: 프리미엄 $2.99/월
- 예상 수익: 월 50~200만원

### 리스크
- ⚠️ 크롤링 = 쇼핑몰 업데이트 시 깨짐
- ⚠️ 제휴 승인 필요
- ⚠️ 혼자 하기엔 벅참

### 왜 5순위?
- ✅ 수익성 좋음
- ✅ 시장 수요 있음
- ❌ 기술 난이도 높음
- ❌ 유지보수 지옥 (크롤링)
- → 2~3개 앱 완성 후 도전

## 🎯 추천 개발 순서 (기술 학습 흐름)

```
1️⃣ 뽀모도로 RPG (2025.10~12)
   └─ Flutter 기초 + Hive (로컬 DB)
   └─ 서버 없이 앱 완성 경험
   
2️⃣ 타임블록+트리거 (2026.01~03)
   └─ 여전히 서버 없음 (로컬)
   └─ 복잡한 UI/UX 경험 (캘린더, 드래그앤드롭)
   └─ 뽀모도로와 기술 스택 동일 → 빠른 개발
   
3️⃣ 지오펜싱 아바타 (2026.04~07)
   └─ Firebase 입문 (Auth, Firestore)
   └─ 백그라운드 서비스 + 위치 API
   └─ 서버 비용 발생 시작
   
4️⃣ 가족 다이어리 (2026.08~12)
   └─ Firebase 심화 (Storage, 실시간 동기화)
   └─ 이미지/영상 업로드
   └─ 초대 시스템, 그룹 관리
   
5️⃣ 쇼핑 스크랩북 (2027~)
   └─ 크롤링 + 외부 API 연동
   └─ 가장 어려움, 협업 권장
```

### 기술 학습 흐름도

```
[로컬 앱]                    [서버 앱]
    │                           │
뽀모도로 ──→ 타임블록 ──→ 지오펜싱 ──→ 가족앱 ──→ 쇼핑앱
    │           │              │          │         │
  Hive      캘린더 UI      Firebase   Storage   크롤링
  기초 UI    복잡 UI       Auth       실시간     API
                          위치 API   이미지     백엔드
```
